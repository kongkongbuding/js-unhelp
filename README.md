# js-help
 * js 常用方法文档
 
## Development

      npm install
      npm run dev

## Use

      npm install js-unhelp

      // 基础类使用
      import help from 'js-unhelp'

      // 特殊类使用
      import 类名 from 'js-unhelp/dist/类名'



|特殊类名称|说明|
|----|----|
|calc|点线面计算|
|canvas|画布|
|coordinateUtils|经纬度坐标系转换|
|download|下载|
|resize|dom大小变化|
|tree|树结构处理|
|upload|上传|
